  1. Devise a strategy for filling in all of the missing values in the dataset To fill in the missing values of the variable “steps”, I propose to take the mean of the steps for the respective 5-minutes interval, accross all days. Once one NA is filled out, it becomes valid information for the subsequent calculations.

  2. Create a new dataset that is equal to the original dataset but with the missing data filled in.

dataNew <- data
for (i in 1:nrow(dataNew)){
        if (is.na(dataNew[i,"steps"])){
                sub <- subset(dataNew, dataNew$interval==dataNew[i, "interval"])
                dataNew[i, "steps"] <- mean(sub$steps, na.rm = TRUE)
        }
}
